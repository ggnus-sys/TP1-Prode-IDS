# TP1-Prode-IDS
Repo para el TP1 de backend

Primero hay que iniciar un enterno virtual de python e instalar los paquetes con:

`pip install flask requests flask-cors mysql-connector-python  werkzeug`

Crear la base de datos ejecutando en la terminal:

`python init_db.py`

e iniciar la aplicación con:

`flask --app app.py run`
